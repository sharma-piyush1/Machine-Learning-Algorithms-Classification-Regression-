# Machine Learning Algorithms (Classification & Regression)

This repository contains clean, modular, and beginner-friendly implementations of popular Machine Learning algorithms under two main categories:

- **Classification**
- **Regression**

Each algorithm is implemented using both **Jupyter Notebook (.ipynb)** for step-by-step learning and **Python script (.py)** for production-style execution.
